Analisador de Tráfego Web 🌐

Bem-vindo ao projeto! Aqui vamos entender de forma didática como um pacote de internet viaja quando você acessa um site.

O que tem dentro de um "Pacote" de dados?

Quando você acessa um site (como o GitHub), seu computador não envia o site inteiro de uma vez. Ele quebra as informações em pequenos "pacotes". Veja como eles são formados por dentro:

Camada de Rede (IPv4): Aqui fica o "endereço de entrega". O IP de origem (você) e o IP de destino (o servidor do site).
Camada de Transporte (TCP): Aqui ficam as "portas". A porta 80 é usada para tráfego HTTP (texto normal) e a porta 443 é para HTTPS (tráfego criptografado e seguro).
